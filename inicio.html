Aqui estamos aprendendo a utilizar as 03 formas da base CSS

Pelo elemento, pelo ID e por classes(CLASS).

elemento =>h1{ color:blue}; ID=>#titulo{color: red;} Classes=>.texto{font-size: 15px;}

Pela elemento, todos os elementos por ex o H1 terão o mesmo estilo

Pelo ID, apenas aquele elemento com o respectivo ID tera o estilo especificado

Por classes, podera ser repetido em qualquer elemento

Obs: o ID e a CLASSE são incorporados dentro da abertura do elemento
